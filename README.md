# positivity-journal
# 🌙 みつばち日記 (iOS Notes Connect / Home Screen Edition)

iPhoneのホーム画面ショートカット機能に完全対応した、毎晩の「3つの良いこと」記録ツールです。

## 📱 ホーム画面アプリ（PWA）化のための仕様
- `index.html` 内の `apple-touch-icon` 設定により、リポジトリ内の `IMG_4277.png` をそのままアプリアイコンとしてホーム画面に配置できます。
- フルスクリーンモードに対応しており、ホーム画面から起動した際は通常のブラウザのURLバーやメニューが非表示になり、ネイティブアプリのように美しく表示されます。

## 📁 構成ファイル
- `index.html` : アプリ構造 ＆ iOSショートカット用メタデータ
- `style.css` : 夜間用のハニーネイビーUI
- `script.js` : データ保存・コピー・iOSメモ起動ロジック
- `IMG_4277.png` : アプリアイコン用画像

## ⚙️ 初期設定と登録方法
1. iPhoneのSafariで公開されたURLにアクセスします。
2. 共有メニューから「ホーム画面に追加」を選択して登録します。
3. アプリ内の「最初の1回だけ設定」に、iPhoneメモ帳から取得したiCloud共有リンクを保存すれば、次回からホーム画面アイコンをタップするだけの最小ステップで日記の追記・ストックができるようになります。
